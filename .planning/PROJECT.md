# Portfolio Website Redesign

## What This Is

A personal portfolio website with a clean, modern design inspired by Nous Research. Three focused pages: an archive of PM writing ("Arthur's PM Homework"), a minimal experience/CV page, and a reverse-chronological current projects/thoughts page. Homepage redirects to the Now/Thoughts page.

## Core Value

A portfolio that showcases work and thinking without over-engineering — clean structure, minimal friction, self-aware tone.
