Replace existing index.astro with redirect to /now using meta refresh:

- Create simple HTML document with meta refresh tag redirecting to /now
- Use 0-second delay for instant redirect: &lt;meta http-equiv="refresh" content="0; url=/now"&gt;

- Minimal structure: just &lt;head&gt; with meta tags and &lt;body&gt; with fallback message
- Title: "Arthur Pfalzgraf - Portfolio"

Do NOT use JavaScript redirect (window.location). Do NOT use Astro redirects config (meta refresh is simpler for static site). Meta refresh is supported by all browsers and works without JavaScript.

Structure:
