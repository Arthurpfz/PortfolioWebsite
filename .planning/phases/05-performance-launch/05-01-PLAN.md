  <name>Task 2: Add theme initialization script to BaseLayout</name>
  <files>src/layouts/BaseLayout.astro</files>
  <action>
Add inline blocking script in BaseLayout head (before any styles) to prevent flash of unstyled content:

1. Check localStorage for 'theme' key
2. If no stored preference, check system preference via matchMedia
3. Set data-theme attribute on document.documentElement immediately
4. Script must be inline (not external) to execute before first paint

Pattern:
```html
<script is:inline>
  const theme = localStorage.getItem('theme') ||
    (window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light');
  document.documentElement.setAttribute('data-theme', theme);
</script>
```

Place script as first child of <head> to ensure it runs before CSS loads. Use is:inline directive to prevent Vite bundling.</action>
